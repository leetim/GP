
### Сборка проекта ###
Для сборки проекта требуется версия *CMake* не ниже 3.0

Сборка проекта *GP*:
  1. Создать директорию temp или с другим произвольным названием.
  2. Использовать команду
  '''bash
  cmake ..
  '''
  3. Вызвать
  '''bash
  make
  '''
  4. Для запуска тестов необходимо ввести
  '''bash
  make test
  '''
